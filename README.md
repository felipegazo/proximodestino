# Project Description
This HTML and CSS project creates a landing page for a travel agency called "Next Destination". The page features a background image, a hero section with text, an offers section, and a contact section.

## Technologies Used
- HTML
- CSS

## Project Structure

#### The project consists of the following files:

- index.html: The main HTML file for the project.
- style.css: The CSS file that styles the HTML elements.
- images/img1.jpg: A background image that is used in the hero section.
- images/Wave.jpg: A wave image that is used at the bottom of the page.
- images/Favicon.png: The website's favicon.
